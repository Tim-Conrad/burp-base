# A BurpSuite Extension Basis

This repository is a starting point for my extensions.

## Included:

- [x] Logging (with different verbosity levels)
- [x] Activations per tool 
- [x] Multiple tabs (Options, Debug, Help)

## Works best with:

- Netbeans 8.1
- BurpSuite Free 1.6.30
- Java OpenJDK 1.7
- Maven
